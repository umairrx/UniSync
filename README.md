# UniSync

A smart course scheduling system for students. Create your perfect weekly timetable with drag-and-drop ease.

## Features

- **Course Management**: Add courses manually or import from JSON
- **Visual Scheduling**: Drag courses onto a weekly grid or double-click slots
- **Conflict Prevention**: Automatic credit tracking and slot management
- **Export**: Save your timetable as a PNG image
- **Data Import/Export**: Backup and restore your schedules
- **Reference Images**: Upload university timetables for side-by-side comparison
- **Customizable Settings**: Adjust schedule times and intervals
- **Themes**: Light and dark mode support

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Radix UI components
- DnD Kit for drag-and-drop
- IndexedDB for local storage

## Development

